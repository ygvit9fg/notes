# Git commands:

## База  
git init - начать слежку от гита в той папке в которой ты ввел эту комманду  
git branch -посмотреть какие ветки есть на проекте  
git branch <имя> - создать ветку  
git checkout - переключаться с ветки на ветку, с коммита на коммит или файл  
git cherry-pick - переносить определенный комит из одной ветки на другую без всей истории коммитов  
git merge - перенести все коммиты с одной ветки на другую  
git status - узнать какие файлы сейчас не закомиченны  
git commit - закомитить файл  
git commit -m "любой текст" - закомитить файл с комментарием  
git add <имя файла> - добавить файл для комита  
git add . - добавить все файлы в текущей директории  
git log - список коммитов которые были сделаны на ветке на которой вы находитесь  

## Чтобы вернуться назад:

git reset:  
'1. git reset --soft <хэш коммита>' - удаляет коммиты но оставляет изменения всех коммитов в виде готового к пушу коммита (типо как после git add)  
'2. git reset --hard <хэш коммита>' - удаляет коммиты и чистит всне их изменения оставляя код коммита до хэш которого ты указал  
 
git revert <хэш коммита> - создает коммит, который отменяет изменения указанного
