#Конспект по командам линукса

## База

1. **pwd** - показывает в какой папке я нахожусь  

2. **ls (list)** - показыавет какие файлы есть в той папке где ты находишься  
   *ls также принимает параметры:*    
   **ls -p** - показать все файлы и отельно выделить папки символом "/" в конце названия папки *(folder/)*  
   **ls -la** - показывает все файлы в текущей директории включая все скрытые файлы  

3. **cd <путь>** - перемешение в директорию  
   *cd также можно использовать кратким способом:*  
   cd ../ - переход на уровень(директорию) выше  
   cd ~ - вернуться в домашнюю папку  

4. **mkdir <имя>** - создание *папки* в текущей директории  
5. **touch <имя>** - создание *файла* в текущей директории  
6. **rm <имя>** - удалить *файл* (в текущей директории, если не указан путь)   
   **rm -rf <имя>** - удаление *папки* со всем содержимым  
7. **cat <файл>** - вывести содержимое файла в терминал  

## Права доступа  

**sudo(SuperUser Do)** - выполняет команды от имени администратора  

**chmod , chown** *(выполняется с sudo):*  

**chmod <права> <файл>** - изменение прав доступа (например chmod 764 test.txt означает что у админа есть право запускать, изменять и смотреть файл, у группы есть право изменить и просмотреть файл, а у пользователя есть права только посмотреть)
то есть:  
7 (админ/владелец): rwx(чтение, запись , выполнение)  
6 (группа): rw-(чтение,запись)  
4 (остальные/все остальные): r--(чтение) 

"*7: rwx , 6: rw- , 4: r--*"   

**chown <group.name:name> <файл>** - изменяет имя группы и создателя (пример - sudo chown frogbee:frogbee notes.md(сделает пользователя frogbee владельцем файла, если вдруг этот файл принадлежал root(то есть был создан через sudo)))
 
## Перемещение  

1. **mv <откуда> <куда>** - переместить файл куда нужно и вдобавок при перемещении можно переименовать
2. **cp <что> <куда>** - скопировать файл        
   **cp -r** - для копирования папок
3. **grep <текст> <файл>** - поиск текста внутри файла  

## Устновка приложения / пакеты 

**Вместо apt-get в новых версиях можно просто apt!**  

*Все загрузки и удаления делаются от имени администратора (с sudo)*  

**sudo apt-get install <название приложения>** - скачать приложение  
**sudo apt-get remove <название приложения>** - удалить приложение   

**sudo apt-get upgrade** - обновить  пакеты всех установленных приложений     
**sudo apt update** - скачать последние версия срверов (без этого линукс может скачать старые версии)   

## Поиск информации о приложениях  

**apt-cache policy <название приложения>** - узнать статус приложения (скачено на моей устройстве или нет)   
**apt-cache search <первые буквы приложения>** - поиск всех приложений по ключевым буквам (например если забыли название программы bluefish, то можно ввести blue а дальше консоль найдем все слова с "blue")
